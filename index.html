핵심 요구사항 분석

  • AI 창작 보조 — 프롬프트 기반 스토리 생성(GPT-4o), 문맥 기반 일러스트 자동 생성(DALL-E 3/Stable Diffusion), 4종 스타일(수채화/크래프트/라인 드로잉/포인트 스티커), 맞춤법 교정
  • 단계별 창작 위자드 — 첨부 플로우 이미지 분석 결과, 이야기 씨앗 찾기→씨앗 심기→물주기→줄기 만들기→스토리보드→삽화 생성→꾸미기의 9단계 가이드 흐름 확인
  • 그림책 캔버스 에디터 — 드래그 앤 드롭 페이지 구성, 템플릿 시스템, 텍스트 편집(폰트/색상/크기), 실시간 자동 저장
  • 출판 및 다국어 — 책 넘김 효과 미리보기, PDF 출판 포맷 변환, 다국어 번역 지원(Phase 2)
  • 커뮤니티 — 사용자 프로필/내 서재, 그림책 갤러리 공유·열람, 소셜 로그인(Google, Kakao)
  • 관리자 — 콘텐츠 관리, API 사용량 통제, 회원 관리

제안 포인트

1. 리치 에디터 구축 경험 — Lexical 기반 170파일 규모 에디터(26종 커스텀 노드, 40+ 플러그인)를 구축한 경험으로, 그림책 캔버스 에디터의 복잡한 인터랙션(드래그 앤 드롭, 레이어 관리, 템플릿 적용)을 안정적으로 구현합니다.

2. AI API 연동 상용 서비스 경험 — ChatGPT API를 활용한 AI 투자 보고서 자동 생성 기능을 상용 서비스로 구축·운영한 경험이 있으며, 프롬프트 엔지니어링 설계부터 API 비용 최적화까지 실전 노하우를 보유하고 있습니다.

3. PDF 문서 처리 파이프라인 — 7중 PDF 처리(HTML→PDF 변환, OCR, 양식 채우기, 병합 등) 파이프라인 구축 경험으로, 그림책의 디지털/인쇄용 출판 포맷 변환을 효율적으로 구현합니다.

4. 첨부 플로우 이미지 기반 설계 — 제공해 주신 "나만의 그림책 만들기" 사용자 플로우(9단계 위자드)를 상세 분석하여, 이야기 씨앗 찾기부터 꾸미기까지의 단계별 UX를 충실히 반영합니다.

PRD (Product Requirements Document)

프로젝트 요구사항을 정의하고 기능 범위를 명확히 합니다.

1. 프로젝트 배경 및 목적

사용자가 그림을 직접 그리지 않아도, 글을 전문적으로 쓰지 못해도 누구나 쉽게 자신만의 그림책을 만들 수 있는 환경을 제공합니다. 생성형 AI(텍스트·이미지)를 활용하여 스토리 아이디어를 얻고, 텍스트에 어울리는 일러스트를 자동 생성하며, 웹 기반 에디터로 직관적으로 페이지를 구성합니다. 완성된 그림책을 출판 포맷으로 변환하고, 사용자 간 공유 커뮤니티를 구축합니다.

2. 사용자 정의

사용자 유형설명주요 기능
일반 사용자 (창작자)그림책을 만들고 싶은 비전문가AI 창작 위자드, 캔버스 에디터, 출판, 내 서재, 갤러리 공유
일반 사용자 (열람자)다른 사용자의 그림책을 감상하는 독자갤러리 열람, 책 넘김 뷰어, 프로필 조회
관리자플랫폼 운영자콘텐츠 관리, 회원 관리, API 사용량 모니터링

3. 기능 요구사항

3.1 AI 창작 보조 기능

  • [필수] 프롬프트 입력 기반 스토리 아이디어 및 전개 제안 (GPT-4o API)
  • [필수] 페이지별 문맥 기반 일러스트 자동 생성 (DALL-E 3 / Stable Diffusion API)
  • [필수] 다중 일러스트 스타일 선택 (수채화, 크래프트, 라인 드로잉, 포인트 스티커)
  • [필수] 맞춤법 검사 및 문맥 교정
  • [선택] AI 생성 결과 재생성/변형 옵션

3.2 단계별 창작 위자드 (첨부 플로우 기반)

  • [필수] 이야기 씨앗 찾기 — AI 프롬프트 입력 → 스토리 아이디어 카드 생성
  • [필수] 이야기 씨앗 심기/물주기 — 선택한 스토리 구조화·발전·교정
  • [필수] 이야기 줄기 만들기 — 페이지별 텍스트 확정
  • [필수] 스토리보드 — 페이지 썸네일 기반 시각적 구성 (체크박스 진행 표시)
  • [필수] 동화삽화 만들기 — AI 일러스트 생성 + 4종 스타일 선택

3.3 그림책 캔버스 에디터

  • [필수] 드래그 앤 드롭 페이지 구성 (텍스트, 이미지 배치)
  • [필수] 그림책 템플릿(레이아웃) 선택 및 적용
  • [필수] 텍스트 폰트, 색상, 크기 편집
  • [필수] 실시간/주기적 자동 저장

3.4 출판 및 다국어 지원

  • [필수] 책 넘김 효과 미리보기
  • [필수] PDF 출판 포맷 변환 및 다운로드
  • [Phase 2] ePub 변환
  • [Phase 2] 다국어 번역 지원

3.5 커뮤니티 및 공통 기능

  • [필수] 소셜 로그인 (Google, Kakao) + 이메일 회원가입
  • [필수] 사용자 프로필 및 내 서재
  • [필수] 그림책 갤러리 공유 및 열람
  • [선택] 좋아요/북마크

3.6 관리자 기능

  • [필수] 전체 콘텐츠 관리, 회원 관리
  • [필수] AI API 사용량 모니터링 및 통제
  • [Phase 2] 상세 통계 대시보드
